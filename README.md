# 🌍 PLANET

## 📌 소개

캘린더 기반 SNS 서비스
사용자는 일정을 기록하고, 이를 선택적으로 공개하여 다른 사용자와 공유할 수 있습니다.
다이어리와 소셜 네트워크의 성격을 결합한 서비스입니다.

---

## ✨ 핵심 기능

### 🚀 MVP

* 일정 생성 및 관리 (CRUD)
* 일정 공개 / 비공개 설정
* 다른 사용자 일정 열람

### 🔮 확장 기능 (Planned)

* 팔로우 기능
* 공개 범위 세분화 (전체 / 팔로워 / 비공개)
* 채팅 기능
* 일정 공유 및 참여 기능
* 일정 기반 인터랙션 (좋아요, 댓글 등)

---

## 실행 방법
### Frontend
- npm install (package 설치)
- npm run dev (빌드 없이 실행)
- npm build .

### Backend
- go run ./cmd/server/main.go (빌드 없이 실행)
- go build ./cmd/server (빌드)

### docker (postgreSQL)
- docker compose up -d
#### 삭제
- docker compose down -v
